# vibe-ui

Beautiful UI effects for vibecoders. Browse the gallery, copy the command, ship faster.

**[Browse the gallery](https://weildthesword.github.io/vibe-ui/website/)**

## Quick Start

```bash
npx vibe-ui fetch gradient-wave
```

Then tell your AI: *"use the code in ./gradient-wave/index.html"*

## Commands

```bash
npx vibe-ui fetch <key>       # Download a component
npx vibe-ui list               # List all components
npx vibe-ui search <query>     # Search by name or tag
```

## Components

### Text Animations
- `gradient-wave` — Animated gradient flowing through text
- `typewriter-deluxe` — Multi-phrase typewriter with blinking cursor
- `glitch-text` — Cyberpunk glitch with chromatic aberration
- `text-reveal-blur` — Words fade in from blur with staggered timing

### Background Effects
- `particles-drift` — Floating particles with connecting lines
- `gradient-mesh` — Morphing mesh gradient (Stripe/Vercel style)
- `aurora-glow` — Northern lights shimmer effect
- `noise-grain` — Subtle film grain texture overlay

### Page Transitions
- `fade-slide-up` — Staggered fade-and-slide entrance
- `curtain-reveal` — Split-curtain opening animation
- `stagger-cards` — Cards cascade in one-by-one

### Hover Effects
- `magnetic-button` — Button pulls toward cursor
- `glow-border` — Animated gradient border trace
- `tilt-card` — 3D tilt following cursor with light reflection

### Loading Animations
- `skeleton-pulse` — Shimmer loading placeholders
- `spinner-orbit` — Three dots orbiting in a circle

### Scroll Effects
- `scroll-reveal` — Elements animate in on scroll
- `parallax-hero` — Background parallax depth effect

## How It Works

1. **Browse** the [gallery](https://weildthesword.github.io/vibe-ui/website/) to find an effect
2. **Fetch** it with `npx vibe-ui fetch <key>`
3. **Ship** — tell your AI coding agent to use the downloaded file

Every component is a self-contained HTML file with zero dependencies. Open it in a browser to preview, or hand it to your AI.
