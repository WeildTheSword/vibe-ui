How the parallax works

The starfield background behind the hero moves at roughly half the speed of your scroll, creating a layered depth effect. The foreground text and buttons scroll normally while the cosmos drifts slowly behind.

CSS Layer

The background is absolutely positioned and oversized to allow vertical movement without gaps. A CSS gradient plus radial-gradient dots create the starfield without any images.

JavaScript Enhancement

A scroll listener applies a translateY transform to the background layer, moving it at 40% of the scroll speed. requestAnimationFrame ensures silky-smooth updates locked to the display refresh rate.

Production ready

This component is self-contained with zero external dependencies. The parallax effect degrades gracefully on devices that do not support smooth scrolling. On mobile, the effect is applied conservatively to prevent performance issues.

Customization

Adjust the parallax intensity by changing the speed multiplier in the script (currently 0.4). Swap the starfield for any pattern, gradient, or even a blurred image to match your brand.
