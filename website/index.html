How It Works

Three steps from browsing to shipping.

1

Browse

Scroll through live previews of text animations, background effects, page transitions, and more. Find exactly the effect you want.

2

Fetch

Copy the terminal command and run it. Or use npx vibe-ui fetch <key> to download the component source code into your project.

3

Ship

Tell your AI coding agent "use the code in this file" and it builds your UI with the exact effect. Pixel-perfect, every time.
